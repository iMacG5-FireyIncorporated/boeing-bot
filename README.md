# FireyBot Open Source Edition
### User Guide
#### How to link source files to bot:
* Enter your token in:
client.token('');
#### How to use certain commands:
* Polls:
+poll;Question;Ans1;Ans2
* Windows 95 KeyGen:
+win95kg;retail/oem
* Random Number Generator: 
+randnum int/flt
* Do I have x Role?:
+doihaverole;rolename
